**File Key:** `D1Ua0kJKF7MVDiSjjVFqjf`

## Instructions

1. Ensure the localhost server is running: `python3 -m http.server 8080` (from the v3 directory)
2. Open `figma-import.html` in your browser for a better interface with Copy buttons
3. Or click any capture link below to open the page with the Figma capture toolbar
4. Select viewport size in the capture toolbar:
   - **Desktop:** 1440px width
   - **Mobile:** 390px width
5. Click "Capture" to send to Figma

**Important Notes:**
- Each capture link is **single-use** - once captured, the link expires
- To capture both desktop AND mobile versions, capture one viewport, then the toolbar will auto-generate a new capture ID for the next capture
